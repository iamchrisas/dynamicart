# LAB | DYNAMICART

## Summary

A lightweight app to showcase your art

- You can
  - create an account, login, logout
  - view, create, update, delete posts
  - view, create, update, delete tags
  - view a user's posts

 User ==> Posts => Tags

                    +---------+ 
                    |  User   |
                    +---------+
                    | UserID  |
                    | Username|
                    | Email   |
                    | Password|
                    +---------+

                    +---------+
                    |  Post   |
                    +---------+
                    | PostID  |
                    | UserID  |
                    | Title   |
                    | Content |
                    +---------+

                    +---------+
                    |  Tag    |
                    +---------+
                    | TagID   |
                    | Name    |
                    | PostID  |
                    +---------+
